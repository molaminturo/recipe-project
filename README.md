# 🍲 Recipe Finder Website

## 📖 Overview
Recipe Finder is a responsive website built using HTML and CSS. It is designed to display different recipes in a clean and organized layout, helping users easily browse food ideas and ingredients.

This project was created to strengthen my skills in front-end design and responsive web development.

---

## 🚀 Features
- 🍽️ Clean and simple recipe layout  
- 📱 Responsive design for mobile and desktop  
- 🖼️ Organized recipe cards with images  
- 🎨 User-friendly and modern interface  
- 🧭 Easy navigation through sections  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3
